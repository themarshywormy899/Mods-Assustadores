# Como instalar?
(No PC) Aperte as teclas Windows + R e escreva %appdata%, aperte OK e em seguida clique na pasta ".minecraft". Jogue todos os arquivos .jar dentro da pasta "mods". 
Agora ja pode ir jogar!
